# Sarkari Photo Date Maker (Free Online Tool)

A free, client-side tool to add **Date of Photo (DOP)** and **Name** on passport size photos for SSC, UPSC, and Railways exams.

## Features
- ✅ Auto Resize to 20kb - 50kb (Strict Mode)
- ✅ Add Name & Date Strip automatically.
- ✅ No Login Required.
- ✅ 100% Privacy (Runs in Browser).

## How to use
1. Upload Photo.
2. Enter Name and Date.
3. Click Download.
